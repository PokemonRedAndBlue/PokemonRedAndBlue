## Controls
- Movement / navigation:
  - Arrow keys: directional input.
  - R: Reset game state (re-initializes player, trainer, and animations).
  - Y / T: Cycle tiles.
  - O / P: Switch trainer sprite variants (Prev/Next)
  - Escape: Exit the game
